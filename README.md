# 🚀 TechSprint – ProductivAI

**ProductivAI** is a smart productivity tracking Android application built with **Flutter** that helps users understand and improve their daily productivity using **Google Calendar insights** and intelligent scoring.

The app analyzes meetings, schedules, and focus time to generate a **Productivity Score**, along with actionable tips to improve work-life balance.

---

## ✨ Features

- 🔐 **Secure Authentication**
  - Google Sign-In (OAuth 2.0)
  - Firebase Authentication
  - Email/Password Login (limited features)

- 📊 **Productivity Score**
  - Score out of 100 based on daily meetings
  - Color-coded productivity indicators
  - Smart productivity messages & tips

- 📅 **Google Calendar Integration**
  - Fetch today’s meetings
  - Analyze meeting duration & frequency
  - Open Google Calendar directly from app

- 💡 **Smart Tips Engine**
  - Personalized productivity tips
  - Recommendations based on your score

- ☁️ **Cloud Powered**
  - Firebase Authentication
  - Google Calendar API
  - Secure token handling

- 🎨 **Modern UI**
  - Minimalist dark gradient UI
  - Smooth animations
  - Clean, professional design
